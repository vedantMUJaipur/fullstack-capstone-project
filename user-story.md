# User Stories for Gift Shop Application

## Story 1: View Gifts
* **As a** Visitor
* **I want to** view a list of all available gifts
* **So that** I can browse what is for sale.

## Story 2: Search Gifts
* **As a** Registered User
* **I want to** filter gifts by category or name
* **So that** I can find exactly what I need quickly.
